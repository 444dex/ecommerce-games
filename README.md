# Ecommerce Games

Protótipo de um **e-commerce de Video Games** desenvolvido como projeto de estudo/prática para criação de uma loja online.

> Este projeto é um protótipo funcional focado em conceitos de e-commerce (catálogo de jogos, carrinho de compras, filtros etc.) e interação com o usuário — ideal para portfólio ou base para evolução futura.

---

## Features

- Listagem de produtos (Video Games)
- Carrinho de compras interativo
- Navegação e filtros básicos
- Layout responsivo
- Estrutura pensada para fácil expansão

---

## Tecnologias

O projeto foi desenvolvido utilizando:

| Camada | Tecnologia |
|------|------------|
| Backend | PHP |
